Why UniMarket?

Built for real-world scale

JWT Authentication

Stateless, secure token-based auth with bcrypt password hashing and 1-hour token expiry.

Redis Caching

Cache-Aside pattern for product listings. Automatic invalidation on write — blazing-fast reads.

PostgreSQL + Supabase

Relational data modeling with raw SQL joins, foreign keys, and cascade delete logic.

Image Uploads

Multer-powered file uploads with disk storage and static asset serving via Express.

Rate Limiting

Global (100 req/15min) and per-route (5 products/hour) rate limits to prevent abuse.

Winston Logging

Structured JSON logs with timestamps, request tracing, and a global error-handling middleware.
